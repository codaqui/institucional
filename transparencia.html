Os saldos são atualizados em tempo real pelo backend. Para contribuir, acesse a página de Apoio. A API completa está documentada em https://api.codaqui.dev/docs.

Como funciona

Usamos contabilidade de dupla partida: cada doação cria dois lançamentos — um débito na conta de entrada (Stripe Income) e um crédito na carteira virtual da comunidade escolhida. Isso garante que nenhum valor apareça ou desapareça sem rastreamento completo.

O backend NestJS valida cada pagamento diretamente com a API da Stripe via Webhook antes de registrar qualquer transação. Pagamentos não confirmados nunca chegam ao sistema.
